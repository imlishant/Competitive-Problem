# 771. Jewels and Stone

Difficulty :: Easy

```cpp
class Solution {
public:
    int numJewelsInStones(string J, string S) {
        map<char, int> count;
        for(char c : S){
            count[c]++;
        }
        int sum = 0;
        for(char c : J){
            sum+=count[c];
        }
        return sum;
    }
};
```
